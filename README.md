## 🚀 Featured Project: Wasi_aL_OS v1.6
A colorful terminal-based mini operating system simulation for Termux.

## 📥 Installation Guide
Copy the whole command below, paste it into Termux, and press **Enter**. (Takes 4-5 minutes)

```bash
pkg update -y && pkg install git python -y && git clone [https://github.com/WasiaL007/Wasi_aL_OS-v1.6.git](https://github.com/WasiaL007/Wasi_aL_OS-v1.6.git) && cd Wasi_aL_OS-v1.6 && clear && python run.py
